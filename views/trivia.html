Which code editor or IDE do you use?
Microsoft's Visual Studio Code. It's free! And quite fast given that it's built with Electron.js. The theme I use is called "Dark Korokai". For Java I've recently switched from Intellij IDEA to Eclipse since it's way less resource demanding. I'm going to try (again) to learn and use Vim in my spare time in the future.
Any upcoming programming languages you're excited for?
Yes, Crystal lang. Both the syntax and performance are great. The motto/slogan is "Fast as C, slick as Ruby".
